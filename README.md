# InfnetFood

## Instalação

1. Clone o repositório:

   ```bash
   git clone https://github.com/luizgabrielduarte/InfnetFood.git
   ```

2. Acesse a pasta do projeto:

   ```bash
   cd InfnetFood
   ```

3. Abra o projeto na IDE de sua escolha.

4. Instale as dependências:

   ```bash
   npm install
   ```

5. Inicie a aplicação:

   ```bash
   npx expo start
   ```

6. Escolha onde executar o aplicativo:

   * Pressione **w** para abrir na Web.
   * Pressione **a** para abrir no Android.
   * Escaneie o QR Code com o **Expo Go** para executar no celular.
